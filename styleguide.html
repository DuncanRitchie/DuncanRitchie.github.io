Code-blocks

My blog (as well as my Childhood Blog in its archived state) includes code-blocks, <pre><code></code></pre>. These have syntax highlighting, courtesy of Shiki which comes built into Astro (the static-site generator I use).

Articles in my blog are written in Markdown, so the syntax for a code-block is as follows: 

```html
<p>This is HTML code.</p>
```

Articles in my Childhood Blog do not contain code. The code-blocks are on a page rendered by an .astro file, using a component built into Astro: 

<Code code="<p>This is HTML code.</p>" lang="html" theme="slack-dark" />

(Astro’s <Code> component seems not to inherit the Shiki theme from the project settings, so it must be specified for each code-block.)
